# Richfield FC Website

A 5-page static football club website built with HTML and CSS.

## Pages

- **index.html** – Home page
- **about.html** – Club history and information
- **contact.html** – Contact details and enquiry form
- **fixtures.html** – Upcoming matches and results
- **players.html** – Player profiles and squad information

## Features

- Navigation bar with links to all pages
- Embedded image assets (club logos, match photos)
- Footer with club details

## Getting Started

Open `index.html` in any modern web browser. No server or build step required.

## Technologies Used

- HTML5
- CSS3
